# QProject
Distributed application based on Celery and RabbitMQ

## Requirements

1. Python 3.5
2. Celery 4
3. RabbitMQ 3.6

## Gateway installation and running
```
$ pip install -r requirements.txt
$ python manage.py migrate
$ python manage.py runserver
```
